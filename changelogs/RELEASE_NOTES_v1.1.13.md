# Forge v1.1.13

## Highlights

- refined the status bar so information hierarchy reads more cleanly at a glance
- improved long-list picker behavior so selection stays centered in a visible window
- added list position feedback for command and model pickers

- clearer grouped status-bar labels for model, effort, agent, tokens, secrets, and memory
- smarter truncation in the top bar so layout holds up better on narrower terminals
- smoother navigation in long picker lists with above/below overflow indicators
- better selected-row readability in long scrolling popup lists
